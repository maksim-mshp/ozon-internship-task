# Укорачиватель ссылок

Решение тестового задания на стажировку в Ozon Банк.

**Автор:** Чернышков Максим

## Запуск в Docker

```bash
docker compose up -d --build
```

## Локальный запуск

```bash
make up
```

## Переменные окружения

Для настройки окружения нужно создать `.env` по аналогии с `.env.example` и поменять значения.

## Тестирование

```bash
make test # unit-тесты
make test-integration # интеграционные тесты
make coverage # покрытие тестами
```

## Линтер

В качестве линтера на backend используется `golangci-lint`.

```
make lint
````

## Генерация OpenAPI-документации

```bash
make openapi
```

Результат генерации будет в папке `backend/api`.
